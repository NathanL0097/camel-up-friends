# 骆驼快跑 · 好友房

一个采用原创网页界面、支持 2–8 人实时联机的骆驼竞速桌游 MVP。房主创建房间后，把链接发给朋友即可加入，无需注册。

## 本地启动

需要 Node.js 20 或更新版本：

```bash
npm install
npm start
```

然后打开 <http://localhost:3000>。同一 Wi-Fi 下，可把 `localhost` 换成这台电脑的局域网 IP 后分享给朋友。

## 已实现

- 6 位房间码和邀请链接，最多 8 人
- 断线后通过浏览器本地身份自动重连
- 5 匹比赛骆驼及逆向奔跑的黑白疯狂骆驼、叠驼移动、灰骰特殊规则
- 5/3/3/2/1 彩色赛段投注牌、绿洲/海市蜃楼板块、终局冠军/末名预测
- 全房间同步的骰子落地、骆驼移动动画，以及赛段冠军高亮反馈
- 随机起跑公告、约一秒的移动观察间隔，以及房主赛后再来一局
- 每种颜色仅能使用一次的秘密终局卡，按正确预测先后顺序结算
- 带玩家颜色和昵称首字的大号赛道板块
- 内嵌于椭圆跑道中央的一屏操作台，低高度窗口无需上下滚动操作
- 服务端统一规则判定与实时状态同步
- 手机和电脑响应式界面

## 公开部署

这是一个普通 Node.js Web 服务，可部署到 Render、Railway、Fly.io 等支持 WebSocket 的平台。启动命令为 `npm start`，平台需提供 `PORT` 环境变量。当前房间保存在内存中，服务器重启后房间会消失；正式长期运营建议接入 Redis 或数据库。
